How to add a new update (no coding required):
  1. Make a copy of post-welcome.html and rename it, e.g. post-2026-budget.html.
  2. Open it in any text editor and change the title, date, and body text.
  3. Back here on updates.html, copy one of the cards above, update its title/date/excerpt, and point the link to your new file.
  4. Save, commit to GitHub, and it’s live. Delete this yellow note before launch.
Want a point-and-click editor instead of editing HTML? You can later connect a free static CMS like Decap CMS — ask your web helper to set it up.
